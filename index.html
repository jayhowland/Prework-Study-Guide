CSS

  • A margin indicates how much space we want around the outside of an element.

  • text-align, will designate where youe text will be aligned per the setting of the page.

  • href is used to specify the location of those outside sources by using a relative path to connect the based code to the current location od assocaited files or sources.
  • card is a classs selctor designation.
  • a period that proceeds a class selector designates it as a clas selctor, and any html element assigned this clas will hold all associated css properties.
  • width selects the desired width of an elelment
  • margin selects the desired height and width by designating two values
  • border designates a desired style of border by assinging a specific size and color and opaqueness.
